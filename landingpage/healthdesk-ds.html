Sticky Stack Pattern

Each card is position:sticky at the same top value with increasing z-index. Non-last cards get margin-bottom:44vh as scroll distance. The JS listener adds .buried when a later card reaches the sticky position.

/* HTML */ <div class="stack-container"> <div class="feat-card">...card 1...</div> <div class="feat-card">...card 2...</div> <div class="feat-card">...card 3...</div> </div>
/* JS — paste once, after DOM ready */ const STICKY_TOP = 96; const cards = document.querySelectorAll('.feat-card'); window.addEventListener('scroll', () => { cards.forEach((card, i) => { const next = cards[i + 1]; if (!next) return; card.classList.toggle( 'buried', next.getBoundingClientRect().top <= STICKY_TOP + 2 ); }); }, { passive: true });
⚠️ Parent sections using sticky cards must not have overflow:hidden or overflow:clip — either will silently break position:sticky.

Vanta Fog Config

Load three.js r134 and vanta.fog before calling. Colors are tuned to match the teal palette.

/* Scripts (CDN) */ <script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r134/three.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/vanta@0.5.24/dist/vanta.fog.min.js"></script> /* Init */ VANTA.FOG({ el: "#vanta-bg", mouseControls: true, touchControls: true, gyroControls: false, minHeight: 200, minWidth: 200, highlightColor: 0x00b4bc, /* --teal-vivid */ midtoneColor: 0xe4f6f7, /* light mist */ lowlightColor: 0x008c97, /* --teal-ui */ baseColor: 0xfafffe, /* near-white */ blurFactor: 0.62, speed: 1.2, zoom: 0.9 });
